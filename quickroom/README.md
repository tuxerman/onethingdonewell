# QuickRoom
A very straightforward browser-based room layout editor. 

## Usage
Open `index.html` in a browser, or serve statically.

## Features
- Drag furniture from sidebar onto canvas
- Resize (8-point handles), rotate (free or snap to 90°), duplicate, delete
- Multiple rooms with edge snapping
- Doors/windows snap to walls
- Text annotations and arrows
- Keyboard: arrow keys to nudge, `R` to rotate, `Del` to delete, `Cmd+D` to duplicate
- Pan (two-finger scroll), zoom (Cmd+scroll or pinch)
- Save/load JSON, export PNG
