# One Thing Done Well

Small tools, each built to do one thing well.

Static site deployed via Cloudflare Pages. [`index.html`](index.html) is the landing page linking out to each tool.

## Tools

- **[QuickRoom](quickroom/)** — browser-based room layout editor. See [quickroom/README.md](quickroom/README.md).

## Adding a tool

1. Create a new subdirectory with its own `index.html` (and `README.md`).
2. Add a `.tool-card` link in the root [`index.html`](index.html).
